ソフトテニス部の部員、練習メニュー管理、AI相談が可能です。

##概要
ソフトテニス部の部員・練習メニューを管理し、練習メニューをAIに相談できるWebアプリです。

##機能
-メンバーの一覧表示・登録・削除
-練習メニューの一覧表示・登録・削除
-練習メニューをAIに相談

##使用技術
-Python(Flask)
-HTML
-SQLite3
-CSV
